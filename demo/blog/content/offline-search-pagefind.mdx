Pagefind builds a static index at the end of the build and loads only the
fragments it needs as the user types.

<Callout type="tip">
  The index is generated from the built HTML, so it always matches what the
  reader actually sees.
</Callout>

## How it fits in

The engine ships the search page; the build script runs Pagefind over `dist`
and writes the index into `public/pagefind`. The result works offline.
