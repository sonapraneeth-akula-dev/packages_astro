# Go Basics

Go programs are organised into **packages**. Execution starts in `package main`
at the `main` function.

## Errors are values

Go has no exceptions. Functions return an `error` alongside their result, and
callers check it explicitly.

```go
f, err := os.Open("notes.txt")
if err != nil {
    return fmt.Errorf("open notes: %w", err)
}
defer f.Close()
```

The `defer` statement schedules cleanup to run when the surrounding function
returns — the Go equivalent of RAII-style teardown.
