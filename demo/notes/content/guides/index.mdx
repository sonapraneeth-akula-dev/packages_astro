# Deep Nesting

The sidebar folds folders into collapsible groups and handles **six levels of
nesting**. This page is **level 1**. Each child folder adds another level — open
them in the sidebar and follow the trail down to level 6.

<Callout type="note">
  Groups are derived from folders; a folder's `index.mdx` provides the group
  label and its own page. The active branch auto-expands.
</Callout>
