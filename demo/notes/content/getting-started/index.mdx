# Getting Started

This section walks you through installing the engine, configuring a site and
writing your first note. Pick a page from the sidebar, or read on.

<Steps>

1. **Install** the core package and Astro.
2. **Configure** your `notes.config.ts` and `astro.config.ts`.
3. **Write** MDX notes under `src/content/docs/`.
4. **Build** with `bun run build` to emit the static site and search index.

</Steps>

Continue to [Installation](/getting-started/installation) to begin.
