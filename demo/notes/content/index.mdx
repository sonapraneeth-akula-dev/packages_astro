# Notes Engine — Demo Handbook

Welcome! This site is a **live showcase** of the `@sonapraneeth/notes-core` engine
in its simplest shape: a **single handbook** with one auto-generated sidebar
that spans every page — no sub-notes.

It walks through installation, configuration, the built-in MDX components and the
performance features — optimised images and offline search — that ship with the
engine.

<Callout type="tip" title="Looking for sub-notes?">
  The companion **notebooks** demo hosts several independent handbooks (C++, Go
  and C#) under one site, each with its own scoped sidebar and a hub home page.
</Callout>
