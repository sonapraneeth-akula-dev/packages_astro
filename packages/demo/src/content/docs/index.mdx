# Notes Engine — Demo Handbook

Welcome! This site is a **live showcase** of the `@grihasetu/notes-core` engine.
It renders a group of MDX notes for a single subject with everything you need
for a great documentation experience:

- A **left sidebar** with an automatically-generated table of contents that
  folds up to **six levels deep**.
- **Per-page contents** ("On this page") plus **previous / next** navigation.
- **Offline search** powered by Pagefind — no server required.
- **Highly optimised images** for top Lighthouse performance scores.
- A full set of **MDX components** shared from the blog: callouts, cards, tabs,
  steps, file trees, badges and syntax-highlighted code.

<Callout type="tip" title="This is the landing note">
  The home page renders `src/content/docs/index.mdx`. Every other note appears
  in the sidebar and gets its own page with breadcrumbs and pager links.
</Callout>
