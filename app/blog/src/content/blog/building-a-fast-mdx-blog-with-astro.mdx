## Syntax highlighting & language detection

Fenced code blocks are highlighted at build time with a high-contrast theme that
clears WCAG AA — even comments. The language is detected from the fence info
string and shown in the frame header.

```ts title="greet.ts"
// Language is detected from `ts` and rendered with a file-name frame.
export function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

## Line numbers

Opt into line numbers per block with `showLineNumbers`:

```py showLineNumbers
def fib(n: int) -> int:
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

## Highlighting a block of lines

Mark specific lines (or ranges) to draw the eye to what matters:

```js {2-4} showLineNumbers
const config = {
  cache: true,        // highlighted
  retries: 3,         // highlighted
  timeout: 5_000,     // highlighted
  verbose: false,
};
```

You can also mark individual lines and ranges together, e.g. `{1, 4-5}`, and use
`ins`/`del` markers for diffs:

```diff
- const slow = await fetchSequential();
+ const fast = await Promise.all(requests);
```
