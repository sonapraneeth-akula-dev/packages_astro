Before reaching for ARIA, reach for the right element. Semantic HTML gives you
keyboard support, focus order and screen-reader meaning at no cost.

## Use landmarks

Wrap your page in `header`, `nav`, `main` and `footer`, and label navigation
regions. Assistive tech turns these into a navigable map.

## One logical heading order

Use a single `h1` per page and never skip levels. The heading tree is how many
users actually navigate long content.

## Label everything interactive

Icon-only buttons need an accessible name, decorative SVGs should be hidden, and
every meaningful image needs alt text.
